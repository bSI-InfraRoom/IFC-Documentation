Semantic definitions at the entity
Entity definition

The IfcNavigationElementType provides the type information for IfcNavigationElement occurrences. A navigation element is an active or passive built element who's primary function is provide navigational instructions and warnings to vessels, this could be in the form of a floating buoy, a fixed beacon or sound signal.
