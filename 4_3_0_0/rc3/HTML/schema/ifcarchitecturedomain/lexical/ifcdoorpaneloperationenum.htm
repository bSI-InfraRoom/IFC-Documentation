Type definition

This enumeration defines the basic ways how individual door panels operate as shown in Figure 296.

Figure 296 — Door operations

The opening direction of the door panels is given by the local placement of the IfcDoor. The positive y-axis determines the direction as shown in Figure 297.

Figure 297 — Door panel operations
