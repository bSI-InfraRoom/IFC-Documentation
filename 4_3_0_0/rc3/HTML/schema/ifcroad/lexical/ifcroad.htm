Semantic definitions at the entity
Entity definition

A route built on land to allow travel from one location to another, including highways, streets, cycle and foot paths, but excluding railways. As a type of Facility, Road provides the basic element in the project structure hierarchy for the components of a road project (i.e. any undertaking such as design, construction or maintenance). NOTE Definition from ISO 6707-1: Way mainly for vehicles. NOTE Definition from PIARC: Line of communication (travelled way) using a stabilized base other than rails or air strips, primarily for the use of road motor vehicles running on their own wheel.
