Semantic definitions at the entity
Entity definition

An IfcOffsetCurveByDistances is a curve defined by a list of offsets along its BasisCurve. If only one offset is provided, it indicates a constant offset along the extents of the basis curve.

Figure 380 illustrates eight instances of IfcOffsetCurveByDistances (in green) defined relative to an IfcAlignmentCurve (in blue).

2OffsetValuesIfcPointByDistanceExpressionL[1:?] List of sequential points described relative to the basis curve. If the offsets do not span the full extent of the basis curve (e.g. if the list contains only one item), then the lateral and vertical offsets implicitly continue with the same value towards the head and tail of the basis curve.
