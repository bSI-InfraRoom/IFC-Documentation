5.2.4.10 Pset_BuildingUseAdjacent

PSET_TYPEDRIVENOVERRIDE / IfcBuilding

Building Use AdjacentProvides information on adjacent buildings and their uses to enable their impact on the building of interest to be determined. Note that for each instance of the property set used, where there is an existence of risk, there will be an instance of the property set Pset_Risk (q.v).
Définition de l'IAI : information sur les bâtiments voisins et sur leur usage pour apprécier leur impact sur le bâtiment auquel on s'intéresse. Veuillez noter que pour chaque instance de ce jeu de propriétés, dès lors qu'un risque existe, il doit exister une instance du jeu de propriétés Pset_Risk (q.v).
