Semantic definitions at the entity
Entity definition

This IfcDistributionElement is a generalization of all elements that participate in a distribution system. Typical examples of IfcDistributionElement's are (among others):

  • building service elements within a heating systems
  • building service elements within a cooling system
  • building service elements within a ventilation system
  • building service elements within a plumbing system
  • building service elements within a drainage system
  • electrical elements
  • elements within a communication network
  • within a sensor (monitoring) network

The IfcDistributionElement is further specialized in the IFC specification. Direct instantiation of IfcDistributionElement without an assigned subtype of IfcDistributionElementType provides the meaning of an distribution element proxy.

HISTORY  New entity in IFC1.5.

IFC4 CHANGE The entity is marked as deprecated for instantiation - will be made ABSTRACT in future releases.
