7.3.4.1 Pset_AudioVisualApplianceTypeCommon

PSET_TYPEDRIVENOVERRIDE / IfcAudioVisualAppliance

Audio Visual Appliance Type CommonAn audio-visual appliance is a device that renders or captures audio and/or video. HISTORY: Added in IFC4.
音響と映像を撮影・録音し放送送出する機器。
