Semantic definitions at the entity
Entity definition

A mobile telecommunications appliance is a device that transmits, converts, amplifies or receives signals used in mobile networks. Note: This entity is used to define specific appliances used in mobile telecommunication networks. General communications appliances and those used in fixed transmission networks should be instantiated using IfcCommunicationsAppliance.
