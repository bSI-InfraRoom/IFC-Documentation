7.7.4.11 Pset_CableSegmentConnector

PSET_TYPEDRIVENOVERRIDE / IfcCableSegment

Properties about cable connectors. This property set is applicable to a type or occurence of IfcCableSegment, indicated that the cable segment has one or two connectors affiliated.
