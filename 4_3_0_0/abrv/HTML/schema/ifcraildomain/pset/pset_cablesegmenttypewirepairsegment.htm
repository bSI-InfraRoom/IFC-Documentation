7.7.4.19 Pset_CableSegmentTypeWirePairSegment

PSET_TYPEDRIVENOVERRIDE / IfcCableSegment / WIREPAIRSEGMENT

Properties of wire pair segments. This property set is applicable to a type or occurence of IfcCableSegment with predefined type WIREPAIRSEGMENT.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
CharacteristicImpedance P_SINGLEVALUE / IfcElectricResistanceMeasure A quantity defined for a mode of propagation at a given frequency in a specific uniform transmission line or uniform waveguide by one of the three following relations:
Z1 = S/ |I|2 Z2 = |U|2 / S Z3 = U / I where Z is the complex characteristic impedance, S the complex power and U and I are the values, usually complex, respectively of a voltage and a current conventionally defined for each type of mode by analogy with transmission line equations.
ConductorDiameter P_SINGLEVALUE / IfcPositiveLengthMeasure Indicates the conductor diameter. It is only used for twisted and untwisted wire pair.
CoreConductorDiameter P_SINGLEVALUE / IfcPositiveLengthMeasure Indicates the core conductor diameter. It is only used for coaxial wire pair.
JacketColour P_SINGLEVALUE / IfcLabel Indicates the colour of the cable or fitting jacket.
ShieldConductorDiameter P_SINGLEVALUE / IfcPositiveLengthMeasure Indicates the shielded conductor diameter. It is only used for coaxial wire pair.
WirePairType P_ENUMERATEDVALUE / IfcLabel / PEnum_WirePairType Indicates the type of wire pair, i.e., twisted, untwisted or coaxial pair.
