7.7.4.58 Pset_ElectricFlowTreatmentDeviceTypeElectronicFilter

PSET_TYPEDRIVENOVERRIDE / IfcElectricFlowTreatmentDevice / ELECTRONICFILTER

Properties associated to electronic filter. An electronic filter is a device designed to transmit spectral components of signals according to a specified law, generally in order to pass the components in certain frequency bands and to attenuate those in other bands (IEC702-09-17)
