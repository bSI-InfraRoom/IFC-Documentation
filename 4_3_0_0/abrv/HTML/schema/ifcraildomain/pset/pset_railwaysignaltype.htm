7.7.4.117 Pset_RailwaySignalType

PSET_TYPEDRIVENOVERRIDE / IfcSignal

Properties common to the definition of occurrences and types of IfcSignal applied in railways.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
SignalIndicatorType P_ENUMERATEDVALUE / IfcLabel / PEnum_SignalIndicatorType Type of the indicators on a signal, e.g. route indicator, speed restriction indicator etc.
LensDiffuserType P_SINGLEVALUE / IfcLabel Type of the lens diffuser the signal is equipped with.
HasConductorRailGuardBoard P_SINGLEVALUE / IfcBoolean Indicates if a guard board is provided.
MaximumDisplayDistance P_SINGLEVALUE / IfcPositiveLengthMeasure The maximum distance that can be displayed. The value relates only to the signal type, not to the circumstances at a special position.
RequiredDisplayDistance P_SINGLEVALUE / IfcPositiveLengthMeasure The required distance that has to be displayed. The value relates only to the signal type, not to the circumstances at a special position.
IsHighType P_SINGLEVALUE / IfcBoolean Indicates if the signal is high (TRUE) or dwarf (ground mounted) (FALSE).
SignalHoodLength P_SINGLEVALUE / IfcPositiveLengthMeasure Nominal length of the signal hood, which is the signal lamp cover against glaring sun.
HotStripOrientation P_SINGLEVALUE / IfcLabel Position of the hot strip, which indicates the direction of the focus of the light beam and is given in terms like "left upper quadrant (LUQ)" or "5 o'clock".
LensDiffuserOrientation P_SINGLEVALUE / IfcLabel Orientation the lens diffuser has to have, which indicates the direction of the lens diffuser and is given in terms like "left upper quadrant (LUQ)" or "5 o'clock".
NumberOfLamps P_SINGLEVALUE / IfcInteger Number of lamps the signal is composed of.
SignalMessage P_SINGLEVALUE / IfcText All possible message available at this signal, e.g. "3/4- display automatic blocking".
RailwaySignalType P_ENUMERATEDVALUE / IfcLabel / PEnum_RailwaySignalType The type of railway signal, e.g. home signal, starting signal, shunting signal, level crossing signal.
