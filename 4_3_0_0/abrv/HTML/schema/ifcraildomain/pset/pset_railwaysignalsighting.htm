7.7.4.116 Pset_RailwaySignalSighting

PSET_OCCURRENCEDRIVEN / IfcSignal,IfcSign

Properties that define information about signal sighting or visibility in railways. These properties are applicable to occurrences of IfcSignal and IfcSign.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
SignalSightingAchievableDistance P_SINGLEVALUE / IfcPositiveLengthMeasure Reading distance of the signal, which is achievable with the help of mitigation works.
SignalSightingAvailableDistance P_SINGLEVALUE / IfcPositiveLengthMeasure Reading distance of the signal without having any mitigation works.
SignalSightingCombinedWithRepeater P_SINGLEVALUE / IfcPositiveLengthMeasure Combined reading distance for the signal and any associated repeaters.
SignalSightingMinimum P_SINGLEVALUE / IfcPositiveLengthMeasure Minimal distance in which the signal has to be readable.
SignalSightingPreferred P_SINGLEVALUE / IfcPositiveLengthMeasure Preferred distance in which the signal shall be readable.
SignalSightingRouteIndicator P_SINGLEVALUE / IfcPositiveLengthMeasure Required reading distance for the route indicator.
SignalViewingMinimumInFront P_SINGLEVALUE / IfcPositiveLengthMeasure Smallest distance where the signal has to be readable (for train very close to the signal).
