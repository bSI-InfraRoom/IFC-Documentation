7.7.4.7 Pset_CableCarrierSegmentTypeCatenaryWire

PSET_TYPEDRIVENOVERRIDE / IfcCableCarrierSegment / CATENARYWIRE

Properties of a catenary wire, which is a longtitudinal wire supporting the grooved contact wires. Properties in this property set are applicable to a type or an occurence ifcCableCarrierSegment with predefined type of CATENARYWIRE.
