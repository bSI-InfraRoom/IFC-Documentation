7.7.4.36 Pset_DataTransmissionUnit

PSET_TYPEDRIVENOVERRIDE / IfcCommunicationsAppliance / MODEM

Properties common to a data transmission unit. This property set is applied to a type or occurrence of IfcCommunicationsAppliance with predefined type MODEM.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
WorkingState P_SINGLEVALUE / IfcLabel Indicates the working state of device or system.
SerialInterfaceType P_ENUMERATEDVALUE / IfcLabel / PEnum_SerialInterfaceType Indicates the type of serial interface used by the device.
DataTransmissionUnitUsage P_ENUMERATEDVALUE / IfcLabel / PEnum_DataTransmissionUnitUsage Indicates the usage of the data transmission unit. It can be used to transmit data for different types of sensors.
