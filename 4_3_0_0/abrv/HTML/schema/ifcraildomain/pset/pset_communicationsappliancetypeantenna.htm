7.7.4.22 Pset_CommunicationsApplianceTypeAntenna

PSET_TYPEDRIVENOVERRIDE / IfcCommunicationsAppliance / ANTENNA

Properties common to an antenna. This property set is applied to a type or occurrence of IfcCommunicationsAppliance with the predefined type ANTENNA.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
AntennaGain P_SINGLEVALUE / IfcRatioMeasure Indicates the antenna gain, which is a ratio of the power transmitted by an antenna in a specific direction compared to an isotropic antenna.
PolarizationMode P_ENUMERATEDVALUE / IfcLabel / PEnum_PolarizationMode Indicates the polarization mode of antenna.
RadiationPattern P_ENUMERATEDVALUE / IfcLabel / PEnum_RadiationPattern Indicates the radiation pattern of antenna.
AntennaType P_ENUMERATEDVALUE / IfcLabel / PEnum_AntennaType Indicates the type of antenna.
