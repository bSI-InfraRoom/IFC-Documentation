7.7.4.140 Pset_TransportEquipmentOTN

/ IfcCommunicationsAppliance / TRANSPORTEQUIPMENT

Properties in this property set are applied to transport equipment that act in optical transport network (OTN) system.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
SingleChannelAveragePower P_SINGLEVALUE / IfcPowerMeasure Indicates the average power of a single channel of the transport equipment.
ChromaticDispersionTolerance P_SINGLEVALUE / IfcReal Indicates the tolerance of the transport equipment chromatic dispersion. The value is defined by picosecond per nanometer (ps/nm).
SingleChannelPower P_BOUNDEDVALUE / IfcPowerMeasure Indicates the power range of a single channel of the transport equipment.
MinimumOpticalSignalToNoiseRatio P_SINGLEVALUE / IfcRatioMeasure Indicates the minimum optical signal to noise ratio of the transport equipment.
PolarizationModeDispersionTolerance P_SINGLEVALUE / IfcTimeMeasure Indicates the polarization mode dispersion tolerance of the transport equipment. It is usually measured by picosecond.
SingleWaveTransmissionRate P_SINGLEVALUE / IfcFrequencyMeasure Indicates the single wave transmission rate of the transport equipment.
EquipmentCapacity P_SINGLEVALUE / IfcIntegerCountRateMeasure Indicates the equipment capacity of the appliance. The value is defined in bits/s.
