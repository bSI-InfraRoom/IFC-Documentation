7.7.4.52 Pset_ElectricalFeederLine

PSET_TYPEDRIVENOVERRIDE / IfcCableSegment / CONDUCTORSEGMENT

Properties of conductors used as feeder line. This property set is applicable to a type or occurence of IfcCableSegment with predefined type CONDUCTORSEGMENT.
