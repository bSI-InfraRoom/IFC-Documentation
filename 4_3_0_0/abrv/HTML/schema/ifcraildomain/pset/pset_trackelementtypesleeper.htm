7.7.4.137 Pset_TrackElementTypeSleeper

PSET_TYPEDRIVENOVERRIDE / IfcTrackElement / SLEEPER

Properties common to the definition to all occurrences and types of IfcTrackElement with PredefinedType set to SLEEPER.
Properties

NameTypeDescription
InstalledCondition P_ENUMERATEDVALUE / IfcLabel / PEnum_InstalledCondition Assessment of the condition of the element at point of installation.

NumberOfTrackCenters P_SINGLEVALUE / IfcInteger Indicates the number of track centers running over the sleepers.
IsHollowSleeper P_SINGLEVALUE / IfcBoolean Indicates whether the sleeper is hollowed or not.
