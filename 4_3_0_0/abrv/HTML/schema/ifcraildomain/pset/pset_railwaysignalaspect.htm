7.7.4.114 Pset_RailwaySignalAspect

PSET_TYPEDRIVENOVERRIDE / IfcSignal,IfcSign

Properties in this property set are applicable for IfcSignal and IfcSign applied in railways. These properties describe the signal aspect, which is the information on the signal or sign shown to the train driver.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
SignalAspectSymbol P_REFERENCEVALUE / IfcExternalReference Content which is shown on the signal or sign, e.g. text, number, arrow or icon.
AppliesToTrainCategory P_ENUMERATEDVALUE / IfcLabel / PEnum_TrainCategory Sign information relative to train category, e.g. freight, passenger.
SignalAspectType P_SINGLEVALUE / IfcLabel The type of aspect, e.g. 2-display aspect for distant signal, 3-display aspect for block signal.
SignLegend P_SINGLEVALUE / IfcText Text information written on the signal or sign.
