Semantic definitions at the entity
Entity definition

An IfcRailway is a spatial structure element as a route from one location to another for guided passage of wheeled vehicles on rails. An IfcRailway acts as a basic spatial structure element that supports to break down a railway project into manageable parts. Note: Definition according to ISO 6706: 2017: national or regional transport system for guided passage of wheeled vehicles on rails.
