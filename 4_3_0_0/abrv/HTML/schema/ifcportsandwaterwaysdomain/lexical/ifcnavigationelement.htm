Semantic definitions at the entity
Entity definition

A navigation element is an active or passive built element who's primary function is to provide navigational instructions and warnings to vessels, this could be in the form of a floating buoy, a fixed beacon. Navigation elements can aggregate other components and elements to form the entire structure. this might include a frame structure to form the body, instances of IfcSign for signage or instances of IfcSignal for supplementary lights an/or sound signals.
