8.17 IfcQuantityResource

8.17.1 Schema Definition

IfcQuantityResource defines a set of basic quantities that can be associated with products through the IfcElementQuantity (defined in IfcProductExtension). The basic quantities are used to assign quantities, which have a particular meaning within a referenced method of measurement.
