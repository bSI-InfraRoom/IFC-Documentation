Semantic definitions at the type
Type definition

This container defines the different predefined types of conveyor segments that can further specify an IfcConveyorSegment or IfcConveyorSegmentType.

Enumeration definition
ConstantDescription
CHUTECONVEYORGravity-operated conveyor where media descends through a trough or chute.
BELTCONVEYORAn endless belt for carrying material without stretching.
SCREWCONVEYORcomposed of a longitudinal screw in a trough or pipe that rotates to force media through the segment
BUCKETCONVEYORA conveyor in the form of connected buckets or segments that move in a continuous loop
USERDEFINEDUser-defined type
NOTDEFINEDUndefined type.
