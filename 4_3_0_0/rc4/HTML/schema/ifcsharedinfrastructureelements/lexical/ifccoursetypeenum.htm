Semantic definitions at the type
Type definition

This container defines the different predefined types of course elements that can further specify an IfcCourse or IfcCourseType.

Enumeration definition
ConstantDescription
ARMOURAn Aggregate layer whose primary function is to protect against erosion of the underlying material by water e.g. riprap. NOTE Definition from ISO 21650: protective layer on a breakwater, seawall or other rubble mound structures composed of armour units
FILTERAn Intermediate layer whose primary function is to prevent the washing through of fine materials.
BALLASTBEDLayer composed of broken stones under the sleepers.
COREA core course is the bulk internal structure of aggregate structures.
PAVEMENTA layer within a pavement structure that forms a paved area or road.
PROTECTIONLayer with the primary task to provide protection against erosion and scour.
USERDEFINEDUser-defined type
NOTDEFINEDUndefined type.
