6.5.4.2 Pset_ElementKinematics

PSET_TYPEDRIVENOVERRIDE / IfcElement,IfcElementType

Information confirming that the element has cyclic and/or pathed kinematic behaviour. The resulting envelope may be availabe as a 'clearance' shape representation.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
CyclicPath P_TABLEVALUE / IfcTimeMeasure / IfcPlaneAngleMeasure Represents the time:angle table of the kinematic behaviour.
CyclicRange P_SINGLEVALUE / IfcPlaneAngleMeasure Identifies the angular range of the kinematic behaviour
LinearPath P_TABLEVALUE / IfcTimeMeasure / IfcLengthMeasure Represents the time:distance table of the kinematic behaviour.
LinearRange P_SINGLEVALUE / IfcPositiveLengthMeasure Identifies the linear range of the kinematic behaviour.
MaximumAngularVelocity P_SINGLEVALUE / IfcAngularVelocityMeasure Identifies the maximum angular velocity of the kinematic behaviour.
MaximumConstantSpeed P_SINGLEVALUE / IfcVelocityMeasure Identifies the maximum constant speed over the kinematic path.
MinimumTime P_SINGLEVALUE / IfcTimeMeasure Identifies the minimum time for the kinematic behaviour.
