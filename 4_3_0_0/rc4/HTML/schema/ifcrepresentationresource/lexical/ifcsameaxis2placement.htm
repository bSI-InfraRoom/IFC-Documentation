IfcSameAxis2Placement

Natural language names
Same Axis2 Placement
Semantic definitions at the function
Function Definition

The function compares two IfcAxis2Placement's and ensures that they are the same (taking an arbitrary epsilon precision factor into account).

HISTORY  New function in IFC2x2
