Maintenace TypeIdentifies the predefined types of maintenance that can be done from which the type that generates the maintenance work order may be set where: ConditionBased: generated as a result of the condition of an asset or artefact being less than a determined value. Corrective: generated as a result of an immediate and urgent need for maintenance action. PlannedCorrective: generated as a result of immediate corrective action being needed but with sufficient time available for the work order to be included in maintenance planning. Scheduled: generated as a result of a fixed, periodic maintenance requirement.
