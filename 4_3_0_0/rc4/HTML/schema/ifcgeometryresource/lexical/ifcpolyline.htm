Semantic definitions at the entity
Entity definition

The IfcPolyline is a bounded curve with only linear segments defined by a list of Cartesian points. If the first and the last Cartesian point in the list are identical, then the polyline is a closed curve, otherwise it is an open curve.

EXAMPLE  Figure 382 illustrates a bounded IfcPolyline and shows the parametric length of each segment and of the total polyline.

Figure 382 — Bounded IfcPolyline with parametric length

NOTE  Definition according to ISO/CD 10303-42:1992
A polyline is a bounded curve of n - 1 linear segments, defined by a list of n points, P1, P2 ... Pn. The _i_th segment of the curve is parameterized as follows:

    for 1 ≤ i ≤ n - 1 where i - 1 ≤ u ≤ i and with parametric range of 0 <≤ u ≤ n - 1.

NOTE  Entity adapted from polyline in ISO 10303-42.

HISTORY  New entity in IFC1.0

ENTITY IfcPolyline
 SUBTYPE OF (IfcBoundedCurve);
  Points : LIST [2:?] OF IfcCartesianPoint;
 WHERE
  SameDim : SIZEOF(QUERY(Temp <* Points | Temp.Dim <> Points[1].Dim)) = 0;
