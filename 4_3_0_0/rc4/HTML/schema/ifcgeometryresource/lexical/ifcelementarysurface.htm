Entity definition

An IfcElementarySurface in the common supertype of analytical surfaces.

NOTE  Definition according to ISO/CD 10303-42:1992
An elementary surface (IfcElementarySurface) is a simple analytic surface with defined parametric representation.

NOTE  Entity adapted from elementary_surface defined in ISO 10303-42.

HISTORY  New entity in IFC1.5
