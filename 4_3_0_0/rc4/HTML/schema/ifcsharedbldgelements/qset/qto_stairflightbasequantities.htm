6.1.5.14 Qto_StairFlightBaseQuantities

QTO_TYPEDRIVENOVERRIDE / IfcStairFlight

Basismengen für alle Bauelemente vom Typ Treppenlauf.
Stair Flight Base QuantitiesBase quantities that are common to the definition of all occurrences of stair flights.
