5.4.4.23 Pset_SpaceParking

PSET_TYPEDRIVENOVERRIDE / IfcSpace / PARKING

Natural language names
Space Parking
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
ParkingUse P_SINGLEVALUE / IfcLabel
Parking UseIdentifies the type of transporation for which the parking space is designed. Values are not predefined but might include car, compact car, motorcycle, bicycle, truck, bus etc.
AccessibleHandicapeIdentifie le type de véhicule pour lequel le parking a été conçu. Les valeurs possibles (voiture, bus, motos, vélos,…) ne sont pas prédéfinies.
駐車場の用途どの車両種別の駐車場なのかを識別します。値はあらかじめ定められないが、自動車、小型車、オートバイ、自転車、トラック、バスなどを含んでいるかもしれません。
ParkingUnits P_SINGLEVALUE / IfcCountMeasure
Parking UnitsIndicates the number of transporation units of the type specified by the property ParkingUse that may be accommodated within the space. Generally, this value should default to 1 unit. However, where the parking space is for motorcycles or bicycles, provision may be made for more than one unit in the space.
TypeVehiculeNombre d'unités du type de véhicule spécifié dans la propriété TypeVehicule que peut contenir l'espace alloué au parking. Généralement, la valeur par défaut est 1. Elle peut être supérieure pour les deux roues.
一区画当たりの駐車台数車両種別ごとの駐車台数のユニットごとの指定台数。一般に、この値は1台/1ユニットになるべきです。しかしながら、オートバイまたは自転車向けである場合、数台/1ユニット以上なるかもしれません。
IsAisle P_SINGLEVALUE / IfcBoolean
Is AisleIndicates that this parking zone is for accessing the parking units, i.e. an aisle (TRUE) and not a parking unit itself (FALSE)
NombreUnitesIndique si cette zone du parking, comme une allée, est réservée à l'accès (VRAI) ou non (FAUX).
通路か駐車スペースの判別駐車場の通路部分(TRUE)か駐車部部分(FALSE)かを示すフラグ。
IsOneWay P_SINGLEVALUE / IfcBoolean
Is One WayIndicates whether the parking aisle is designed for oneway traffic (TRUE) or twoway traffic (FALSE). Should only be provided if the property IsAisle is set to TRUE.
Sens uniqueIndique si cette allée du parking est prévue pour être à sens unique (VRAI) ou à double sens (FAUX). A fournir seulement si la propriété "Est un accès" est égale à VRAI.
一方通行駐車場通路が一方通行(TRUE)か双方向(FALSE)かを示すフラグ。
