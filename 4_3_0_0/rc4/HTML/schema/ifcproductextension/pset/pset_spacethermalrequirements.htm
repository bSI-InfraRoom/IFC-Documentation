5.4.4.26 Pset_SpaceThermalRequirements

PSET_TYPEDRIVENOVERRIDE / IfcSpace, IfcSpatialZone, IfcZone

Property Set Definition in German
Space Thermal RequirementsProperties related to the comfort requirements for thermal and other thermal related performance properties of spaces that apply to the occurrences of IfcSpace, IfcSpatialZone or IfcZone. It can also be used to capture requirements for IfcSpaceType's. This includes the required design temperature, humidity, ventilation, and air conditioning.
室の快適な温熱環境に関連する要求性能。IfcSpace, IfcZoneに適用される。 温度、湿度、空調についての設計上の設定を含む。
