7.4.4.55 Pset_ProtectiveDeviceTrippingUnitTimeAdjustment

PSET_TYPEDRIVENOVERRIDE / IfcProtectiveDeviceTrippingUnit

Protective Device Tripping Unit Time AdjustmentA set of time adjustment values that may be applied to an electronic or thermal tripping unit type.
電磁式または熱動式トリップユニット型に適用することができる時間調整（時延）のセット。
