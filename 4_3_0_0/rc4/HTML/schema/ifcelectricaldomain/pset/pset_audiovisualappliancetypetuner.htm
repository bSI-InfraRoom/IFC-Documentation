7.4.4.10 Pset_AudioVisualApplianceTypeTuner

PSET_TYPEDRIVENOVERRIDE / IfcAudioVisualAppliance / TUNER

Audio Visual Appliance Type TunerAn audio-visual tuner is a device that demodulates a signal into a stream of audio and/or video. HISTORY: Added in IFC4.
