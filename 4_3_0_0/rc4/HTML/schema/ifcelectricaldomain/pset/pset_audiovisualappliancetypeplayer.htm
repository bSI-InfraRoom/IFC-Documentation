7.4.4.6 Pset_AudioVisualApplianceTypePlayer

PSET_TYPEDRIVENOVERRIDE / IfcAudioVisualAppliance / PLAYER

Audio Visual Appliance Type PlayerAn audio-visual player is a device that plays stored media into a stream of audio and/or video, such as camera footage in security systems, background audio in retail areas, or media presentations in conference rooms or theatres. HISTORY: Added in IFC4.
セキュリティーシステムのカメラや店舗などのBGMシステム、または劇場や会議室ないのプレゼンテーションシステムのような収容された音響映像信号を表示・放送する装置。
