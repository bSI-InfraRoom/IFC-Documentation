7.4.4.8 Pset_AudioVisualApplianceTypeReceiver

PSET_TYPEDRIVENOVERRIDE / IfcAudioVisualAppliance / RECEIVER

Audio Visual Appliance Type ReceiverAn audio-visual receiver is a device that switches audio and/or video from multiple sources, including external sources connected from ports and internal aggregated sources. HISTORY: Added in IFC4.
内部や外部に接続された音源を含む複数の情報から、音響と映像信号を切り替える装置。
