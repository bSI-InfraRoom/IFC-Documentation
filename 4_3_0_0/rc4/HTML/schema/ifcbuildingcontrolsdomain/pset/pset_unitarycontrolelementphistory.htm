Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
Temperature P_REFERENCEVALUE / IfcTimeSeries / IfcThermodynamicTemperatureMeasure
TemperatureIndicates the current measured temperature.
Mode P_REFERENCEVALUE / IfcTimeSeries / IfcIdentifier
ModeIndicates operation mode corresponding to Pset_UnitaryControlTypeCommon.Mode. For example, 'HEAT', 'COOL', 'AUTO'.
Fan P_REFERENCEVALUE / IfcTimeSeries / IfcLogical
FanIndicates fan operation where True is on, False is off, and Unknown is automatic.
SetPoint P_REFERENCEVALUE / IfcTimeSeries / IfcIdentifier
Set PointIndicates the temperature setpoint. For thermostats with setbacks or separate high and low setpoints, then the time series may contain a pair of values at each entry where the first value is the heating setpoint (low) and the second value is the cooling setpoint (high).
