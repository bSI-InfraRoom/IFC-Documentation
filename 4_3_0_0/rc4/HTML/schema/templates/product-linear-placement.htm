4.8.2.1 Product Linear Placement

A Product Linear Placement takes into account the ISO 19148 Linear referencing. Each product placed with Product Linear Placement has an absolute placement on the IfcLinearPositioningElement.Representation which is typically instantiated as an IfcAlignment with its appropriate axis geometry. Therefore, similarly to how IfcSpatialElement.ObjectPlacement sets the context for all contained elements, the IfcLinearPositioningElement sets the context for all elements positioned on it. Consequently, each product placement that uses Product Linear Placement references an IfcLocalPlacement which is the placement of an IfcLinearPositioningElement through IfcLinearPlacement.PlacementRelTo.
