Semantic definitions at the type
Type definition

An IfcNumericMeasure is the numeric value of a physical quantity.

Type: NUMBER

NOTE  Type adapted from numeric_measure defined in ISO 10303-41.

HISTORY  New type in IFC1.5.1.
