7.5.4.110 Pset_ValveTypePressureRelief

PSET_TYPEDRIVENOVERRIDE / IfcValve / PRESSURERELIEF

Valve Type Pressure ReliefSpring or weight loaded valve that automatically discharges to a safe place fluid that has built up to excessive pressure in pipes or fittings. Note that a pressure relief valve is constrained to have a single port pattern.
