7.5.4.109 Pset_ValveTypePressureReducing

PSET_TYPEDRIVENOVERRIDE / IfcValve / PRESSUREREDUCING

Valve Type Pressure ReducingValve that reduces the pressure of a fluid immediately downstream of its position in a pipeline to a preselected value or by a predetermined ratio. Note that a pressure reducing valve is constrained to have a 2 port pattern.
