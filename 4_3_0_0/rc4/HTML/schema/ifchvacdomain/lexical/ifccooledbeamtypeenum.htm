IfcCooledBeamTypeEnum

Natural language names
Cooled Beam Type Enum
Enumération des types de poutre froide
Semantic definitions at the type
Type definition

There are two general types of cooled or chilled beams: passive and active. An active Cooled Beam uses a fan or other auxilliary device to aid in air recirculation, while a passive Cooled Beam relies solely on convection to cool the space.

HISTORY  New enumeration in IFC2x2.

Enumeration definition
ConstantDescription
ACTIVEAn active or ventilated cooled beam provides cooling (and heating) but can also function as an air terminal in a ventilation system.
PASSIVEA passive or static cooled beam provides cooling (and heating) to a room or zone.
USERDEFINEDUser-defined cooled beam type.
NOTDEFINEDUndefined cooled beam type.
